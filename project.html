Project Overview

OTTOVOX is an innovative system designed to integrate real-time communication, control mechanisms, and advanced functionalities for hardware and software solutions. Below are the project goals, technologies used, and implementation details.

Project Goals

  • Enable real-time control of hardware using intuitive interfaces.
  • Implement seamless communication between software and hardware modules.
  • Design a user-friendly and visually appealing web interface.
  • Incorporate AI features for predictive analysis and automation.

Technologies Used

Frontend

HTML, CSS, JavaScript

Database

Firebase

Hardware

ESP32, TB6612FNG, OLED Screen, N20 & Servo Motor, MAX98357A, INMP441 MIC, Speaker, SD card and Module

API

Gemini, Whisper, ElevenLabs, Assemblyai

3D Modeling

Bambu Lab, Fusion 360

Implementation Details

The OTTOVOX system is implemented as a hybrid solution combining software and hardware capabilities. Key features include:

  1. Real-time video feed integration using ESP32.
  2. Interactive control panel for managing device movements.
  3. Advanced AI algorithms for automation and predictions.
  4. Responsive web design for seamless user experience.
